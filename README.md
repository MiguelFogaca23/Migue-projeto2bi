# Biblioteca Online

Projeto desenvolvido para a disciplina de Desenvolvimento Web 1.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Node.js
- Express
- PostgreSQL

## Como executar

1. Criar o banco PostgreSQL.
2. Executar o arquivo banco.sql.
3. Instalar dependências:

npm install

4. Iniciar servidor:

node server.js

5. Abrir o index.html no navegador.
